# Bug 管理台（演示）


说明：这是一个静态演示页面，展示了 ID、现象、Bug 分类，并支持通过分类下拉框进行筛选与文本搜索。

快速使用：

1. 在文件管理器中打开 `e:/bug-dashboard/index.html`（或直接在浏览器中用 `file://` 打开）。
2. 使用顶部的搜索框输入关键字，或从下拉菜单选择分类进行筛选。

文件说明：

- `index.html` — 主页面（使用 Tailwind CSS CDN）
- `styles.css` — 少量自定义样式
- `app.js` — 数据与交互逻辑
